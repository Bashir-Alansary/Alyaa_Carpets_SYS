# 🏪 Inventory Management System

A modern web application for inventory and sales management, built using Google Apps Script as the backend and a single HTML frontend.

## ✨ Features

- 📦 Inventory Management
  - Add, edit and delete products
  - Product categories (Types)
  - Low stock alerts
  - Out of stock detection

- 🛒 Sales
  - Create invoices
  - Automatic stock deduction
  - Sales history
  - Search & filters

- 📊 Reports
  - Daily / Weekly / Monthly reports
  - Sales statistics
  - Best selling products
  - Export reports (Coming Soon)

- 👥 Users
  - Admin & Employee roles
  - Login system
  - Permission management

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

2. Create a Google Spreadsheet.

3. Open Google Apps Script.

4. Copy the backend code.

5. Deploy as Web App.

6. Paste the Web App URL inside

